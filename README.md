# House Price Prediction With Linear Regression From Scratch

## Project Goal

The goal of this project is to understand linear regression by building it from scratch using NumPy, then comparing it with scikit-learn's implementation.

This project focuses on the full machine learning workflow:

- loading a real dataset
- understanding features and target variables
- splitting data into training and testing sets
- scaling features
- implementing linear regression manually
- training with gradient descent
- evaluating model performance
- comparing against a library implementation

## Dataset

This project uses the California Housing dataset from scikit-learn.

The task is to predict median house value using numerical features such as median income, house age, average rooms, population, latitude, and longitude.

## Main Concepts

- supervised learning
- regression
- linear models
- loss functions
- mean squared error
- gradient descent
- feature scaling
- train/test split
- model evaluation
